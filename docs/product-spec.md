# Product Spec

## User Journey

1. Open homepage.
2. Enter birth data.
3. Choose Mock Demo or Custom Paipan.
4. Optionally enable LLM narrative and provide BYOK key.
5. Generate life spectrum report.
6. Inspect pillars, palace grid, dimensions, dayun, yearly table, chart, heatmap, raw JSON.
7. Export JSON or Markdown.

## Page Structure

- Hero/input workspace
- Birth input panel
- Provider and key panel
- Generation status
- Summary report
- BaZi pillars
- Ziwei palace grid
- Dimension cards
- Dayun spectrum table
- Spectrum line chart
- Dayun heatmap
- Yearly color scale table
- Raw JSON viewer
- Export actions
- Disclaimers and footer

## MVP

- Mock Demo report generation
- Custom paipan provider with shenjige mapping
- Rule-based scoring
- Optional LLM narrative
- Dashboard and exports

## v1

- Real provider calibration
- Stronger scoring rules and fixtures
- Public deployment

## v2

- PDF export, multilingual UI, history storage, and additional providers.
